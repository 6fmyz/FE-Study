CSS

学习笔记3

1. 文本：text-indent 属性实现文本缩进；text-align 影响一个元素中的文本行互相之间的对齐方式；word-spacing 属性可以改变字（单词）之间的标准间隔；letter-spacing修改的是字符或字母之间的间隔；text-transform 属性处理文本的大小写；text-decoration 属性是文本装饰；white-space 属性会影响到用户代理对源文档中的空格、换行和 tab 字符的处理；direction 属性影响文本方向。
2. 字体：font-family 属性 定义文本的字体系列；font-style 属性最常用于规定斜体文本；font-variant 属性可以设定小型大写字母；font-weight 属性设置文本的粗细；font-size 属性设置文本的大小。
3. 链接：a:link - 未被访问的链接，a:visited - 用户已访问的链接，a:hover - 鼠标指针位于链接的上方，a:active - 链接被点击的时刻；a:hover 必须位于 a:link 和 a:visited 之后，a:active 必须位于 a:hover 之后；text-decoration 属性大多用于去掉链接中的下划线；background-color 属性规定链接的背景色。
4. 列表：要修改用于列表项的标志类型，可以使用属性 list-style-type；想对各标志使用一个图像，这可以利用 list-style-image 属性。

HTML

学习笔记2

1. style:background-color 属性为元素定义了背景颜色；font-family、color 以及 font-size 属性分别定义元素中文本的字体系列、颜色和字体尺寸；text-align 属性规定了元素中文本的水平对齐方式。
2. 链接：href 属性规定链接的目标；使用 Target 属性，你可以定义被链接的文档在何处显示；name 属性规定锚（anchor）的名称、创建 HTML 页面中的书签。

概述

学习笔记1

1. HTML：超文本标记语言，标准通用标记语言下的一个应用。
2. CSS：层叠样式表(英文全称：Cascading Style Sheets)，是一种用来表现HTML或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。
3. JavaScript：一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。常用来为网页添加各式各样的动态功能,为用户提供更流畅美观的浏览效果，广泛用于客户端。
4. SVN：Subversion的简称，是一个开放源代码的版本控制系统，用于多个人共同开发同一个项目，共用资源的目的。Subversion将文件存放在中心版本库里，这个版本库很像一个普通的文件服务器，不同的是，它可以记录每一次文件和目录的修改情况，这样就可以借此将数据恢复到以前的版本，并可以查看数据的更改细节。
5. Git：一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。
